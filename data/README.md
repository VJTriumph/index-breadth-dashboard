# Data Folder

Upload your 4 index CSV files here.

## Expected CSV format

Each CSV file should have a column with stock symbols. Example:

```
Symbol
RELIANCE
TCS
INFY
HDFCBANK
```

## Files to upload

- `nifty50.csv` — Nifty 50 stocks
- `banknifty.csv` — Nifty Bank stocks
- `niftyit.csv` — Nifty IT stocks
- `midcap150.csv` — Nifty Midcap 150 stocks

Once uploaded, the dashboard will automatically read stocks from these files.
